# GoHighLevel → Mailchimp Contact Sync Automation

This project automates the synchronization of contacts from GoHighLevel to Mailchimp using Make.com.

It eliminates manual data entry by automatically adding contacts to Mailchimp whenever a new contact is created in GoHighLevel.

---

## 🚀 Overview

This automation captures contact data from GoHighLevel and processes it to:

- Watch for new contacts in GoHighLevel
- Search for existing subscribers in Mailchimp
- Prevent duplicate entries
- Add new subscribers to Mailchimp
- Ensure accurate and real-time synchronization

---

## ⚙️ How It Works

1. GoHighLevel triggers when a new contact is added
2. Mailchimp searches for existing subscriber by email
3. System checks if contact already exists
4. New subscribers are added to Mailchimp
5. Existing subscribers are skipped to avoid duplicates
